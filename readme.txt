== Description ==
 
Je náš plugin pro Wordpress, který zpracovává požavky na stránky, kde je plugin nainstalován. Jen požavky robotů vyhledávačů (aktualizovaný seznam je níže) se posílají do našeho systému [www.semor.cz](https://www.semor.cz) kde jsou zpracovávány do přehledných grafů a tabulek.
Dozvíte se tak například, jak často váš projekt procházejí jednotliví roboti, jestli nedostávají chybový status (404,500 atd.). Do systému se nepřenášejí žádné citlivé informace a vše je v souladu GDPR.

== Installation ==
= From within WordPress =

1. Visit 'Plugins > Add New'.
2. Search for 'SEMOR SeachBot'.
3. Activate SEMOR in your plugins page.
4. Follow the steps on "after activation" below.

= Manually =

1. Upload the `semoranalyzer` folder to the `/wp-content/plugins/` directory.
2. Activate the SEMOR Analyzer plugin through the 'Plugins' menu in WordPress.
3. Follow the steps on "after activation" below.

= After activation =

1. Click on Settings > SEMOR SearchBot.
2. Save TOKEN for your web project from SEMOR
3. You're done! You should see the data coming into your SEMOR.cz account.

If you have any problems, write our support team.
